# Number Trainer
![Logo](Icon.png)
Simple arithmetic skills trainer.

## Develops
- Concentration
- Instant memory
- Quick count in the mind

## Installation
```
go install fyne.io/fyne/v2/cmd/fyne@latest
git clone https://github.com/mandriota/numtrain
cd numtrain
fyne install
```
